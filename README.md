cdd
===

A simple Bash script to create shortcuts for paths in terminal.

How?
----

1) source /path/to/cdd.sh

2) Type: cdd

Config
------

Shortcuts are stored in ~/.cdd

Usage
-----

cdd add <shortcut> <path>
cdd remove <shortcut>
cdd list [-v]
cdd <shortcut>
